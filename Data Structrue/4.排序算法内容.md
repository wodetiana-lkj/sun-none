# 1.冒泡排序

基础

```java
for(int i = 0; i < arr.length ; i++) {
    for(int j = i+1 ; j < arr.length ; j++) {
        if( arr[i] > arr[j] ) {
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }
    }
}
```

基础便于理解基本排序，但是并不符合冒泡定义

```java
for(int i = 0; i < arr.length-1 ; i++) {
    for(int j = arr.length-1; j > i ; j--) {
        if(arr[j] < arr[j-1]) {
            int temp = arr[j];
            arr[j] = arr[j-1];
            arr[j-1] = temp;
        }
    }
}
```
